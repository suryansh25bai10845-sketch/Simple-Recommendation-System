# Recommendation System (Python) 
## Project Overview: - 
This project is a simple recommendation system made using Python.
The main idea behind this project is to understand how recommendation 
systems work in real life.
It helps in suggesting content to users based on their choices. 
The system gives recommendations for movies, songs, web series, and 
videos.
It is a basic project made for learning and improving programming 
skills. 
## Features: - 
This system allows the user to choose diƯ erent options like category, 
language, and type of content.
Based on these choices, it shows suitable recommendations. 
The categories include movies, songs, web series, and videos. 
The language options are Hindi and English. 
The types include funny, thriller, horror, and family. 
## How It Works: - 
- The program works using simple data stored in Python dictionaries.

- It asks the user to enter their preferences. After taking input, it matches the data 
and shows the results. 

- This is not a machine learning system. 

- It is a basic logic-based system, which makes it easy to understand for beginners. 
## Technologies Used: - 
This project is made using Python. It uses basic programming concepts like 
dictionaries, conditions, and user input. 
## How to Run: - 
To run this project, you need to have Python installed on your system. Open 
the file in any Python editor and run it.
After running, enter your choices when the program asks, and it will show recommendations.
